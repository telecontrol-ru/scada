# Debugger
{:.no_toc}

* TOC
{:toc}

The debugger window is intended for diagnosing protocol-level data
exchange. It is available when the Client is started with the
command-line option `--debug`.

The window displays a request table with these columns:

| Column | Description |
|---|---|
| ID | Request identifier |
| Phase | Request status |
| Start time | Time when the request was sent |
| Duration | Execution time in milliseconds |
| Description | Request description |

When a request row is selected, the full contents of the request and
response are shown.

The number of displayed requests is limited to 1,000.
