# User configuration

The users window can be opened from the main menu with `Further ->
Users`.

![]({{ '/img/users.png' | relative_url }})

Double-clicking a user row starts editing of the user name or access
rights.

Available access levels:

| Access level | Control functions and setpoints | Configuration editing |
|:---|:---:|:---:|
| 0 - Executive / viewer | No | No |
| 1 - SCADA engineer | No | Yes |
| 2 - Dispatcher | Yes | No |
| 3 - Administrator | Yes | Yes |

To create a new user, choose `Create -> User` from the window's context
menu.

To delete a user, choose `Delete` from the user context menu.

To change a password, choose `Set password` from the user context menu.
