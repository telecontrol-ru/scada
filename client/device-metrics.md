# Метрики устройств
{:.no_toc}

* TOC
{:toc}


Окно метрик отображает ключевые показатели производительности для одного или нескольких устройств в виде сетки. Строки соответствуют метрикам, определенным в типе устройства, столбцы — устройствам.

Для вызова окна метрик выберите устройство или направление в панели оборудования и нажмите кнопку *Метрики* на панели команд, либо выберите пункт *Метрики* из контекстного меню. При выборе направления отображаются метрики по всем устройствам, подключенным к направлению.

Значения метрик обновляются в реальном времени.
