# Отладчик
{:.no_toc}

* TOC
{:toc}


Окно отладчика предназначено для диагностики протокольного обмена данными. Отладчик доступен при запуске Клиента с параметром командной строки `--debug`.

Окно отображает таблицу запросов с колонками:

| Колонка | Описание |
|---------|----------|
| ID | Идентификатор запроса |
| Фаза | Статус запроса |
| Время начала | Время отправки запроса |
| Длительность | Время выполнения запроса (мс) |
| Описание | Описание запроса |

При выделении строки запроса отображается полное содержимое запроса и ответа.

Количество отображаемых запросов ограничено 1000.
