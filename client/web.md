# Веб-интерфейс
{:.no_toc}

* TOC
{:toc}

Помимо графического Клиента для настольных систем, ОИК предоставляет веб-интерфейс для доступа к системе через браузер. Веб-интерфейс предназначен для удаленного доступа и работы с тонких клиентов.

## Возможности

Веб-интерфейс предоставляет основные функции работы с системой:
* Авторизация и аутентификация
* Просмотр дерева объектов и оборудования
* Отображение данных в таблицах и сводках
* Управление и ручной ввод значений
* Квитирование и просмотр событий
* Задание уставок
* Экспорт данных в CSV
* Работа с избранным
* Конфигурирование объектов и устройств

## Ограничения

Следующие функции доступны только в графическом Клиенте для настольных систем:
* **Графики и тренды** — визуализация данных в виде графиков
* **Мнемосхемы Модус** — отображение мнемосхем ActiveXeme
* **Интеграция с Vidicon** — поддержка протокола Vidicon
* **Полноценная печать** — печать с предварительным просмотром

## Запуск

Веб-сервер запускается как отдельное приложение (`webserver`). Параметры сервера (порт, адрес) задаются аргументами командной строки.

После запуска веб-интерфейс доступен через браузер по адресу сервера.

## Интерфейс

Веб-интерфейс использует стиль Bootstrap и включает:
* Строку главного меню с теми же пунктами, что и в графическом Клиенте
* Панель инструментов с кнопками команд
* Рабочую область с поддержкой вкладок и панелей
* Панели, прикрепляемые к краям окна (снизу, сверху, слева, справа)
* Строку состояния

Разметка окон и рабочих листов сохраняется аналогично графическому Клиенту.
