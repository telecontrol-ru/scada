# Данные
{:.no_toc}

* TOC
{:toc}


Окно `Данные` отображает историю изменения значений выбранного объекта в виде таблицы с колонками:

| Колонка | Описание |
|---------|----------|
| Время | Временная метка источника (с точностью до миллисекунд) |
| Значение | Значение объекта |
| Качество | Признак качества значения |
| Время получения | Время получения значения сервером |

Щелчок по заголовку первого столбца переключает порядок сортировки записей по времени (по возрастанию или убыванию).
