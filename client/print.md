# Печать
{:.no_toc}

* TOC
{:toc}

Содержимое окон с данными (таблицы, сводки) может быть распечатано с предварительным просмотром.

## Вызов печати

Для печати содержимого активного окна нажмите кнопку *Печать* на панели команд или выберите соответствующий пункт контекстного меню.

## Предварительный просмотр

Перед печатью отображается окно предварительного просмотра, позволяющее:
* Просмотреть макет печати
* Настроить параметры принтера
* Выбрать ориентацию страницы

Данные форматируются в виде таблицы с заголовками столбцов.

## Печать таблиц и сводок

При печати таблиц и сводок форматирование выполняется автоматически:
* Заголовки столбцов отображаются в первой строке
* Данные размещаются в ячейках таблицы
* Формат адаптируется под размер бумаги
