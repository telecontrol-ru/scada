# Пользовательская таблица
{:.no_toc}

* TOC
{:toc}

Пользовательская таблица — это окно с электронной таблицей для мониторинга и отображения данных в произвольном табличном формате. Столбцы обозначаются буквами (A, B, C...), строки — числами (1, 2, 3...).

Для создания пользовательской таблицы выберите меню *Таблица — Новая пользовательская таблица*.

## Строка формул

В верхней части окна располагается строка формул. При выделении ячейки в строке формул отображается ее содержимое. Для редактирования ячейки следует выделить ее и ввести текст или формулу в строке формул, либо дважды кликнуть по ячейке.

Для подтверждения ввода нажмите *Enter* или *Tab*.

## Ячейки

Ячейки могут содержать:
* Текстовые значения
* Ссылки на объекты данных (формулы) — значения обновляются в реальном времени
* Математические выражения

При привязке ячейки к объекту данных ее значение обновляется автоматически при изменении данных на сервере.

## Режим редактирования

Редактирование ячеек доступно только в режиме редактирования, который включается администратором. В режиме редактирования из контекстного меню доступны команды форматирования.

## Форматирование

Для выделенного диапазона ячеек можно задать:
* **Цвет фона** — задается из контекстного меню
* **Выравнивание** — по левому краю, по центру, по правому краю
* **Условное форматирование** — изменение оформления ячейки в зависимости от ее значения

## Мигание ячеек

Ячейки, значения которых изменились, могут мигать для привлечения внимания оператора.
