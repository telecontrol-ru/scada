# Портфели
{:.no_toc}

* TOC
{:toc}

Портфель — это именованная коллекция объектов данных, объединенных пользователем для удобства мониторинга и анализа. Портфели отображаются в виде иерархического дерева: корневые узлы — портфели, дочерние — включенные в них объекты.

## Создание портфеля

Для создания нового портфеля выберите команду *Создать — Портфель* из контекстного меню панели портфелей. Имя портфеля генерируется автоматически (Портфель, Портфель 2 и т.д.) и может быть изменено.

## Переименование

Для переименования портфеля выберите команду *Переименовать* из контекстного меню портфеля.

## Добавление объектов

Объекты добавляются в портфель из панели объектов или командой *Добавить объекты* из контекстного меню портфеля.

## Удаление

Из контекстного меню доступны команды удаления отдельных объектов из портфеля или удаления портфеля целиком. При удалении объекта из системы он автоматически удаляется из всех портфелей.
